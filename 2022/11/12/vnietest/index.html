    Behind the Curtain: Learning Occluded Shapes for 3D Object Detection【AAAI2022】

    关键词：
    形状学习策略与不利天气、遮挡和截断等导致的点云质量恶化相结合
    代码：
    https://github.com/Xharlie/BtcDet
    数据集：KITTI
    分析形状缺失的情况
    外部遮挡；信号缺失【这边视为能观察到的，只是丢了】，表现为空相素【我的理解是过于稀疏】；自遮挡，远侧部分被近侧遮挡【每个物体都不可避免地会发生自遮挡导致的形状错失】；

    创新点：
    BtcDet是第一个针对受遮挡影响的对象形状的3D对象检测器
    Complete object shapes= observed objects shapes ∪ the occluded object shapes
    根据估计的占用概率P（OS）进行目标检测；shape occupancy 网络
    Method：
    1确定遮挡区域和信号缺失区域
    2shape occupancy 网络 估计RoI的形状占有率 ；
    3BtcDet 再通过backbone提取点云特征，然后将上述估计的形状占有率P（OS）与点云特征相连，送到RPN生成proposals；
    利用局部几何特征 f与grid特征进行refinement。
    assemble the approximated complete shapes
    假设：
    1大多数前景对象类似于数量有限的形状原型；【处理：利用source point来补全target】
    H(A,B) →源B是否覆盖目标A的大部分区域 最终A形状=源B中得分最高的Top3+目标A原始的点云。源对象来自训练集的其他帧。
    2车辆和骑自行车者，大致对称【处理：借助标记的边界框，镜像补充它们】

    识别observed objects shapes ∪ the occluded object shapes【是在球面坐标系中完成】
    使用均匀间隔的球形网格对点云进行体素化，这样的话，位于任何一个点后面的球面坐标下的体素都是被遮挡的体素。Soc【就是这个S可能会存在很多voxel中】非空体素若有包含形状S则为1，会形成一个occupancy 的概率。
    对于信号缺失的区域，作者使用在深度图，寻找位于有雷达信号和没有雷达信号的边界构成的像素→将其投影到球形体素中构成Ssm

    Estimate the shape occupancy
    GT生成。O（S）有点的=1，没有的=0。
    预测每个目标的P（S）。
    编码非空体素，提取特征，送入shape occupancy 网络。→预测P（OS）
    用了一个占有率网络

    Occlusion-Aware Proposal Refinement【plus】
    将P（OS）【转化到笛卡尔坐标系，位于这个笛卡尔体素中的最大的P ( O S )来作为这个体素的概率】到RPN的各层feature中→feature拼接→ROI pooling→优化的bbox
    我的想法：在小目标检测性能较好，感觉就是那个作者的假设起了作用。【强制小目标是镜像对称的，然后补全】尽管目标被遮挡，但通过前景以及已有先验信息，就能补全behind信息。。。

    Dynamic Hypergraph Neural Networks IJCAI2019

    DHGNN关键点：
    动态超图构建（DHG）和超图卷积（HGC）
    HGNN整个文章关键点：如何生成超边
    一个点取他周围的邻居点K个，形成一条由K+1个点的超边。【此处信息会存在较大冗余，因为一个小区域内，邻居重合度大。是一个改进的点】
    1. DHG动态超图构建
    利用 KNN 方法和 k-means 聚类方法在单个推理过程中分别基于局部【生成超边】和全局特征【生成超图结构】更新超图结构。
    生成超边KNN:每个顶点的 k-1 个最近邻。近邻顶点与顶点中形成一条超边。【这条超边包含了k个点】
    超图结构k-means：每一层的整个特征图欧式距离。每个顶点最近的 S-1 个将被指定为该顶点的相邻超边。【这个地方是不是在已有的超边上进行操作。顺序是先初始化超图的。】
    使用输入特征嵌入来初始化超图结构。特征嵌入随着网络的深入而演变，超边集是动态调整的。

    2. HGC超图卷积
    顶点卷积：【常用池化的方法】变换矩阵来置换和加权超边中的顶点；
    MLP 来生成变换矩阵 T，并使用一维卷积来压缩变换后的特征。
    超边卷积：超边卷积中采用了注意机制，使用多层感知 (MLP) 来生成每个超边的权重分数。输出质心顶点特征计算为输入超边特征的加权和。
    【顶点卷积后→Hyperedge Convolution EdgeConv 将相邻的超边特征聚合为顶点的特征。激活函数再更新。特征嵌入更新顶点特征→构建一个新的超图结构。】

    实验部分：使用的是社交网络的数据集。
